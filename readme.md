## ğŸ› Troubleshooting

### Common Issues

**Extension not working:**
1. Refresh the game page
2. Check if you're on the correct website
3. Disable other game extensions temporarily

**Features not applying:**
1. Click "Apply Configuration" in popup
2. Check for feature conflicts
3. Try resetting to default settings

1. Enable "Performance Mode" in advanced settings
2. Reduce number of active features
3. Lower auto-refresh intervals
