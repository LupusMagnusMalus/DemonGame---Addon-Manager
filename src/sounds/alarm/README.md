# Battle Alarm Sounds

This Sound are used for the battle alerts

## Available Sounds:
- `alarm.mp3` - Standard Alarm sound
- `guitar.mp3` - The original guitar sound

## Format:
- Format: MP3
- Recommended Duration: 2-5 seconds
- Recommended Quality: 128kbps or higher

## Usage:
This sounds are being used by the Battle Alert System (base/modules/battle-alarm.js).
The user can choose the used sound through the settings popup under "Notifications".
